# webscraping

### stack
- python
- beautifulsoup4
- selenium

### requests
- 웹 서비스와 상호작용하는데 필수적인 도구중 하나
- 웹에서 보내는 http 통신 요청을 받는 모듈

### xpath
- 마크업 언어(html)에서 특정 경로를 나타내기 위한 언어
- 특히 value값을 찾을 때 해당 태그를 통해 상대 경로로 보다 쉽게 찾는 것이 가능

### user agent
- 운영체제, 웹브라우저의 종류나 버전을 제공
- 주로 호환성 문제가 발생했을 때 이를 추가하여 문제를 해결
